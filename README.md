# History & Geopolitics

A personal library of plain-English history and geopolitics pieces.

Website: https://history-geopolitics.netlify.app/

## Library

### Middle East — Israel & Palestine

- **[The Story of Israel](Israel_Book_Easy.html)** — A plain-English history of Israel in 15 short chapters, from ancient times through October 7, 2023.
- **[Benjamin Netanyahu — A Simple Story](Netanyahu_Easy.html)** — The life of Israel's longest-serving prime minister, told in plain English.

## Running locally

[index.html](index.html) is the library hub. Some browsers block iframes when opening files directly, so serve the folder over a local server:

```
python3 -m http.server 8000
```

Then open http://localhost:8000.
